## 作者
**chenxuan**

![dog](https://chenxuan520.oschina.io/chenxuanweb/pic/untitled1.png)
## 制作目的
*为了使用C++搭建轻量级网页*
## 特点说明
- 支持windows以及linux
- 支持get以及post方法
- 支持自定义扩展后台 通过dealaskcpp来快速扩展网页内容
- 支持mysql数据库
## 使用教程
1. git clone https://gitee.com/chenxuan520/server-for-static-web.git
2. chmod 777 ./install.sh
3. ./install.sh
## 制作说明
- windows使用select模型
- linux使用epoll模型
